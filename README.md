### Starting...</h4>
#### Make a clone</h6>
```
git clone git@github.com:giovannalinda/challenge-repository.git
```
### install the dependencies</h6>
```
yarn
# or
npm install
```
### Usage</h3>
```
yarn dev
# or
npm run dev
```
Open http://localhost:3000 to view it in the browser.
